# Freelance Project Tracker

## Overview

This is a Python desktop application developed as part of a mini project.
The goal of the project is to demonstrate a **user-friendly interface** with proper navigation and backend integration.

The application allows users to manage freelance projects and freelancer details in a simple and organized way.

## Features

* Add, edit, and delete projects
* Manage freelancer information
* View project details clearly
* Dashboard with basic statistics
* Data stored using SQLite

## Technologies Used

* Python
* CustomTkinter
* SQLite
* Matplotlib (optional)
